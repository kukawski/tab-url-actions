Tab URL Actions

Help

This extension lets you quickly navigate to a new URL using the current tab URL as a parameter. See configuration below to learn how it works and how to add new items.

Use the Keyboard, Luke. You can add keyboard shortcut for the popup in chrome://extensions at the bottom.

  • Enter in the filter field jumps to the first match.
  • Enter (or primary mouse button) on a match opens the URL in current or new foreground tab depending on the item configuration.
  • Shift-Enter (or middle mouse button) on a match opens the URL in new background tab regardless of the item configuration.
  • Down (or j) cycles through matches forward.
  • Up (or k) cycles through matches backward.
  • Escape on a match jumps back to the filter field.
